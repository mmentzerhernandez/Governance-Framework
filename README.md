# National AI Governance Playbook for Global Majority Governments

A clickable knowledge base (LLM wiki) for resource-constrained Global South governments standing up
credible AI governance in 12–18 months **without passing any new AI law** — using existing
procurement rules, data-protection statutes, and executive circulars.

## View it

Open **[`wiki.html`](wiki.html)** in any browser. It is a single, fully self-contained file
(no external dependencies) that opens onto the Overview "playbook" and lets you click through to:

- **The 6-component Minimum-Viable Governance Package** (Diagnostic Anchor · Procurement Lever ·
  Regulatory Lever · Redress Focal Point · Infrastructure Safeguard · Protection Survey)
- **The 5-phase Implementation Sequence** (Diagnose → Anchor → Protect → Engage → Monitor)
- **7 capability-area hubs** and **26 framework/tool/org pages** (UNESCO RAM, Canada AIA, Brazil
  ANPD, OECD incident taxonomy, India DPI, ILO/Kenya surveys, and more)

## Structure

```
wiki.html        ← the shareable single-file site (generated)
wiki/            ← the source markdown knowledge base (Obsidian-style [[links]])
  overview.md    ← the landing playbook
  concepts/      ← 6 levers, 7 capability-area hubs, cross-cutting themes
  entities/      ← framework / tool / org pages
  sources/       ← the source report summary
CLAUDE.md        ← wiki schema + domain context
build_html.py    ← regenerates wiki.html from the markdown
AI_Governance_Global_Majority_Research.md  ← the single underlying source report
```

## Provenance caveat

The underlying report was compiled during a web-search retriever outage and flags that its
primary-source URLs and some data points are **unverified**. Verify every cited link at the
original institutional portal before formal use.
